## 요트 

### 기능 구현 목록
- [ ] 게임 엔진
  - [ ] 요트가 부표에 도달할 때까지 게속 게임 진행
  - [ ] 매 주기마다 발생하는 모든 이벤트 처리
  - [ ] 이벤트는 자연 발생 이벤트, 깜짝 이벤트, 요트 관련 이벤트 발생
  - [ ] 자연 발생 이벤트가 한 주기에 여러 번 발생한 경우 우선순위에 따라 하나만 선택
  - [ ] 깜짝 이벤트가 한 주기에 여러 번 발생한 경우 우선순위에 따라 하나만 선택
  - [ ] 요트 관련 이벤트의 경우 모두 처리
  - [ ] 한 주기에 발생한 모든 이벤트 처리 후 요트의 속도와 안정성 변경
  - [ ] 요트의 속도에 따라 요트의 현재 positon 변경
- [ ] 
- [ ] 환경 구현(외부 세계) : 자연 환경의 흐름을 관리하는 시뮬레이터
  - [ ] Wind, Current, Wave, Weather를 매 주기 마다 갱신한다(유기적)
    - [ ] 바람 관리
      - [ ] 바람의 세기와 방향을 관리한다.
      - [ ] 바람의 세기는 매초
    - [ ] 조류 관리
      - [ ] 조류의 세기와 방향을 관리한다
    - [ ] 파도 관리
      - [ ] 파도의 높이를 관리한다.
    - [ ] 날씨 관리
      - [ ] 맑음, 흐림 등을 관리한다
    - [ ] 시간(dt)에 따른 바람, 조류, 파도 변화
    - [ ] 랜덤적으로 바람, 조류, 파도 변화
