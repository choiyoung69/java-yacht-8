# 🛥️ 요트 시뮬레이션 게임
요트 동아리 신입생 교육에서 가장 어려운 점은 실제 환경 조건(바람·조류 등)이 항상 바뀌기 때문에 특정 상황을 반복적으로 연습시키기 어렵다는 것이었습니다.
그래서 자연환경 변화와 상황 대응을 게임 형태로 시뮬레이션하여 어떤 상황에서도 빠르게 판단하고 대응할 수 있도록 도와주는 학습용 프로그램을 만들었습니다.

## 🤔 프로그램 개요
이 프로그램은 풍속, 풍향, 조류 등 자연 환경이 지속적으로 변화하는 상황에서,
사용자가 주어진 선택지 중 하나를 선택하며 요트를 조종하는 텍스트 기반 시뮬레이션 게임입니다.
핵심적인 특징은 다음과 같습니다:

### 🌬 자연 환경 시뮬레이션
- 매 주기마다 바람과 조류가 자연적으로 변화합니다.
- 풍속/풍향 변화, 돌풍·약풍·난류 같은 랜덤 이벤트가 발생합니다.
- 조류는 일정 시간 동안 강하게 지속되거나, 일정 주기마다 방향이 바뀌는 주기적 이벤트가 발생합니다.
- 난이도(쉬움/보통/어려움)에 따라 자연 환경의 변동성과 이벤트 발생 빈도가 달라집니다.

### ⚡ 이벤트 기반 진행
- 매 tick(주기)마다 자연 이벤트, 랜덤 이벤트, 요트 내부 이벤트가 발생합니다.
- 자연·랜덤 이벤트는 한 번에 여러 개가 발생할 수 있지만, 우선순위가 가장 높은 하나만 선택됩니다.
- 요트 내부 이벤트(전복 위험, 속도 저하 등)는 여러 개가 동시에 처리됩니다.
- 각 이벤트는 JSON 기반 선택지(정답/오답)와 효과(안정도·속도 변화)를 갖습니다.

### ⛵ 요트 상태 관리
- 요트는 속도, 안정도, 위치(position)를 지속적으로 업데이트합니다.
- 잘못된 선택이 반복되면 안정도가 낮아져 전복하거나, 속도가 0이 되어 게임이 종료될 수 있습니다.
- 올바른 선택을 하면 안정도/속도가 회복되거나 더 빠르게 앞으로 나아갈 수 있습니다.

### 📦 데이터 기반 구조
- 모든 이벤트(자연/랜덤/요트 내부)는 JSON 파일로 관리됩니다.
- 선택지(정답/오답) 역시 난이도별로 JSON에 저장되어, 프로그램은 이를 자동으로 파싱해 UI에 출력합니다.
- 새로운 이벤트를 JSON에 추가하면 즉시 게임에 반영될 수 있도록 설계했습니다.

### 🎮 게임 목표
- 자연의 변화, 랜덤 요소, 요트의 상태를 적절히 조절하며 요트를 전복시키지 않고 최종 목적지(부표)에 도달하는 것이 최종 목표입니다.
